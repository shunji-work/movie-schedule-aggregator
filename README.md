プロダクト概要
映画上映スケジュール管理アプリ

近隣の映画館の上映スケジュールを一括管理し、「今すぐ観られる映画」を見つけやすくするWebアプリケーション。ユーザーの現在地と時刻を考慮した独自のレコメンデーションアルゴリズムで、最適な映画体験を提案します。

主要機能
1. すぐ観る！
現在時刻から90分以内に始まる上映を表示
インテリジェントなおすすめソート（近い60%、早い20%、旬15%、高評価5%）
複数のソートオプション：おすすめ/ランキング順/近い順/早い順
バッジ表示：最速上映/最寄り/高評価/ランキング上位
2. 今日の上映スケジュール
マイシアター登録済み劇場の上映予定を時系列表示
開始時刻までの残り時間をリアルタイム表示
映画の評価（Filmarks）とスクリーン情報
3. 作品から探す
上映中の全映画をグリッド表示
映画選択後、本日の上映時間を近い劇場順に表示
ポスター画像、ジャンル、評価表示
4. 劇場マップ
Google Maps統合で周辺劇場を地図表示
現在地からの距離を自動計算
マイシアター登録機能（タップで追加/解除）
劇場チェーン別の色分け表示
5. マイページ
視聴履歴の記録と管理
映画ごとにメモ・感想を保存
視聴日と劇場情報を記録

フロントエンド

Vite - 高速ビルドツール
React Router - ルーティング
Tailwind CSS - スタイリング
shadcn/ui - UIコンポーネントライブラリ
Lucide React - アイコン
バックエンド・インフラ
Bolt Database - データベース・認証基盤（PostgreSQL）
Row Level Security（RLS）によるセキュアなデータアクセス制御
データベース構造
テーブル
movies - 映画マスタ（タイトル、ジャンル、上映時間、評価、ランキング）
theaters - 劇場マスタ（名前、チェーン、住所、緯度経度）
showtimes - 上映スケジュール（映画・劇場・時刻・スクリーン）
user_favorite_theaters - マイシアター登録
user_watched_movies - 視聴履歴とメモ
特徴的な実装
レコメンデーションアルゴリズム
独自の重み付けスコアリング：

距離スコア（60%） - 現在地からの近さ
時刻スコア（20%） - 上映開始までの時間
ランキングスコア（15%） - 映画の人気度
評価スコア（5%） - Filmarks評価
劇場チェーン別カラーリング
TOHOシネマズ、イオンシネマ、ユナイテッド・シネマなど、主要チェーンごとに専用の色分け表示で視認性を向上。

地理位置情報連携
Google Maps統合で直感的な劇場検索
距離計算による自動ソート
徒歩・車での移動時間を考慮
セットアップ

# 依存関係インストール
npm install

# 開発サーバー起動
npm run dev

# プロダクションビルド
npm run build

VITE_SUPABASE_URL=your_supabase_url
VITE_SUPABASE_ANON_KEY=your_supabase_anon_key
